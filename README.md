Customer Shopping Behavior Analysis

Overview:

This project analyzes customer shopping behavior using Python, SQL, and Power BI. It covers the complete data analytics workflow—from data loading and cleaning to exploratory analysis, SQL-based insights, and dashboard visualization—aimed at deriving actionable business insights.



Dataset:

The dataset contains customer purchase information, including:

Demographics (Age, Gender, Location)

Product details (Category, Item Purchased)

Purchase behavior (Purchase Amount, Frequency, Discounts)

Subscription and shipping information

Review ratings and payment methods


Tools & Technologies:

Python (Pandas, NumPy) – Data cleaning & EDA

SQL (PostgreSQL / MySQL / SQL Server) – Analytical queries

Power BI – Interactive dashboard & visualizations

Jupyter Notebook – Analysis workflow

pgAdmin – Database management


Steps Performed:

Loaded dataset using Python

Performed Exploratory Data Analysis (EDA)

Cleaned and transformed data (null handling, feature engineering)

Created derived columns (e.g., age groups, purchase frequency)

Loaded cleaned data into SQL database

Ran analytical SQL queries to answer business questions

Built a Power BI dashboard for visualization

Created a report summarizing key insights


Dashboard:

The Power BI dashboard highlights:

Revenue trends by gender and age group

Purchase behavior by category and shipping type

Discount and subscription impact on revenue

Customer segmentation insights


Results & Insights:

Identified high-revenue customer segments

Analyzed purchasing patterns across demographics

Evaluated the impact of discounts and subscriptions

Highlighted top-performing products and categories


How to Run:

Clone the repository

Install required Python libraries

pip install pandas sqlalchemy psycopg2-binary


Run the Jupyter Notebook to clean and analyze data

Load the processed data into PostgreSQL using SQLAlchemy

Execute SQL queries from the provided .sql file

Open the Power BI file to explore the dashboard
